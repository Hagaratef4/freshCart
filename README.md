- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

FreshCart is a modern, responsive e-commerce website built with React.js.  
Users can browse products, view details, add to cart & wishlist, and complete checkout.



##Features
- User Authentication (Login / Register)
- Browse products by category
- Product detail pages with images and reviews
- Add to Cart & Cart management
- Wishlist functionality
- Protected routes for account/checkout
- API integration using Axios
- Responsive design (mobile/tablet/desktop)



##Tech Stack
- React.js
- React Router
- Axios
- Context API for state management
- Tailwind CSS / Bootstrap (choose one)
- Formik + Yup (forms & validation)
- React Icons / Font Awesome



## Installation (Development)
1. Clone the repository:
   git clone https://github.com/Hagaratef4/freshCart.git
2. Navigate into project folder:
   cd freshCart
3. Install dependencies:
   npm install
4. Run the development server:
   npm start
